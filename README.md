# OPTOELECTRONICS & SENSING: 光电传感与仿真工程集

[![Tech](https://img.shields.io/badge/核心技术-ZEMAX%20%7C%20COMSOL%20%7C%20MATLAB-blue)](#) [![Hardware](https://img.shields.io/badge/硬件与界面-C%2B%2B%20%7C%20QT%20%7C%20PYTHON-orange)](#) [![Research](https://img.shields.io/badge/学术成果-论文%20%7C%20专利-success)](#)

> 💡 **项目概述**：
> 本工程库包含了光栅光谱仪的完整研发资料（含底层硬件、上位机界面及光路仿真），以及声光晶体、衍射光栅、大气传输与固体激光的多物理场仿真工程，同时收录了本科阶段的竞赛论文、专利及 IBM 量子算法平台的仿真探索。
